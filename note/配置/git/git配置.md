1. 下载git 安装一直下一步就完事了

2. 配置信息

   1. 找到git目录下的 git Bash 打开

   2. 输入用户名和email

      ``` c
      git config --global user.name "Your Name"
      git config --global user.email "email@example.com"
      ```

      

3. 设置仓库

   1. 在需要设置的仓库下右键git Bash

   2. 输入指令：

      ``` c
      git init
      ```
