# HTML

## 一、标签

1. `<p></p>`:该标签被用来放段落，paragraph的缩写

   **注：段落之间有一定的距离（内外边距）**

2. `<hr>`:分割线 会占一整行

   `<hr width="百分比" />`会按百分比显示， 自带居中

3. `<br>`：换行标签，对应单词break
4. `<nobr></nobr>` ：不换行
5. `<center></center>`：居中显示
6. `<button></button>`：按钮
